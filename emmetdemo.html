Emmet Shortcuts for VSCode

Child: >

  • Here is a shortcut called the Child shortcut
  • The command looks something like:
  • nv>ul>li

Sibling: +

  • You can also use the Sibling shortchut which is..
  • An example like: h1+h2+h2
  • which would create h1 and two h2 elements unnested

Multiplication: *

  • You can also perform another shortcut that's Multiplication
  • This would look something like:
  • ul>li*3 - which would create a ul with 3x nested li

Item numbering: $

  • An example of item numbering and creating classes numbered
  • ul>li.item$*5
  • this would create 5x li nested within a ul and each li with a class ranging from 1-5 all named item.

Text: {}

  • Text within a tag
  • If I wanted text within an anchor tag to be displayed I can..
  • a{Click me} - on the otherhand you can also perform li*5>a{Click me}
  • This would create 5 li with nested anchor tags in each with the text Click me.
